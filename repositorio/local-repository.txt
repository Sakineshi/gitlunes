LOCAL REPOSITORY
	Area donde se almacenan los cambios en
	formato BLOB (Binary File Object) mediante
	una referencia identificada con el nombre
	HEAD, cada commit emitido genera un numero
	hexadecimal de 40 digitos conocido como HASH.

COMANDOS

GIT COMMIT	Genera un mensaje para el almacenamiento con notas detalladas
		git commit -m genera el commit con un mensaje
		git commit -amend genera un commit que reemplaza al ultimo
		git commit - amend -m enmienda con un mensaje

GIT STATUS	compara el orking directory con el staging area

GIT DIFF	compara el working directory con el local repository
